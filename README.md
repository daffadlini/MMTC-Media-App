# MMTC Media by Alpha

[<img src="https://raw.githubusercontent.com/ismartcoding/plain-app/main/assets/get-it-on-github.png" alt='Get it on GitHub' height="80">](https://github.com/daffadlini/MMTC-Media-App/releases/tag/release)

Aplikasi ini diperuntukkan untuk Gelar Karya Simulasi Angkatan 21 Sekolah Tinggi Multi Media Yogyakarta. Aplikasi ini dibuat menggunakan bahasa pemrograman Flutter dan aplikasi pembantu FlutterFlow untuk membentuk keseluruhan tampilan aplikasi.

## Fitur

 - Akses semua fitur yang ada di dalam website media.mmtc.ac.id
 - Menyaksikan Live Streaming Gelar Karya Simulasi Angkatan 21 Sekolah Tinggi Multi Media Yogyakarta.
 - Menyaksikan video-on-demand dari kumpulan karya yang dibuat oleh anggota Gelar Karya Simulasi Angkatan 21 Sekolah Tinggi Multi Media Yogyakarta.
 - Membaca berita terbaru yang ditulis oleh Alpha Newsroom
